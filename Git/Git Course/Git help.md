`git help` is a Git command that **shows documentation and usage information** for Git commands. It’s useful if you want to quickly check **how a command works or its options**.

---

### Basic Usage

`git help`

- Shows **general help** about Git commands.
    

`git help <command>`

- Shows help for a **specific command**, e.g., `git help commit`.
    

---

### Examples

1. **General Git help**
    

`git help`

- Lists all common Git commands.
    

2. **Help for a specific command**
    

`git help status`

- Shows detailed information about `git status`.
    

3. **Alternative formats**
    

`git help -w status`

- Opens the help page in a **web browser** (if supported).
    

`git status --help`

- Equivalent to `git help status`.
    

---

### Quick Summary

- **Purpose:** Learn how to use Git commands and options.
    
- **Usage:**
    
    - `git help` → list commands
        
    - `git help <command>` → detailed help for a command
        
    - `git <command> --help` → same as above
        
    - `git help -w <command>` → open in browser
